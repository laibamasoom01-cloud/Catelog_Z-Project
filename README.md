# Catalog_Z

A responsive photo and video gallery website built using HTML and CSS. This project is inspired by the Catalog-Z template and includes multiple pages such as Photos, Videos, About, and Contact.

## Features

* Responsive Navigation Bar
* Photo Gallery Page
* Video Gallery Page
* About Page
* Contact Page with Contact Form
* Team Members Section
* Search Bar UI
* Pagination Design
* Footer with Social Media Icons
* Font Awesome Icons Integration

## Technologies Used

* HTML5
* CSS3
* Font Awesome

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/Catalog_Z.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
